# git-collaboration-input_root
⚫︎入力√とは
数学を学ぶ学生に向けた数学記号・文字でつまずかないためのwebアプリケーション。
学習の円滑化を図って、数学への苦手意識のきっかけを解消する。

⚫︎課題
・数学の授業で知らない記号や文字が解説なしで扱われることがある。
・自分で記号や文字の意味を調べる際に入力すらできないことがある。
・既存のキーボードで変換が出てこない。

⚫︎機能
・わからない記号の意味を解説。（入力方法は文字または絵）
例）シグマ（候補記号を全て出す。候補（URL）タップで意味・読み・例を表示させる。文脈で意味が変わる場合は複数表示する。※ページ移行なし）
①Σ⇨Σは総和の記号です。「シグマ」と読みます。…..
②σ⇨σは標準偏差を表します。（統計学）…… σは置換を表します。（代数学）…..

⚫︎ロジック
入力
ー手書き（canvas）

記号特定
手書き入力はLaTeXに変換

出力
ー既知→辞書：高速・正確（答えがパターン化）
ー未知→AI：補足説明
